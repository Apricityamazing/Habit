# Habit
## What it is
Habit is a cli habit tracker that allows you to track and keep up with positive Habits you wanna keep.
## Why I made it
I got bored, and I asked Claude AI to give me a random programming language and a project to do. This is not ai generated, but I queried Claude to ask specific questions I have, but I mostly stuck to reading the go documentation.
## How to use it
You probably shouldn't use it, it is the first time I have ever written something in Go, but if you would like to try it out, all you need to do is clone the repository and run go build. From there do ./habit help to see its uses.
## Bugs
If you find a bug, feel free to put it into issues, but if you submit a pull request I most likely won't merge it because this is just a personal project. Feel free to fork this project if you would like to expand it!
